This is a program to classify a trianle and calculate if it is right angled or not.
INPUT: 3 double values (sides).
OUTPUT: Triangle classification and boolean rightAngle.

Written in Java and used JUnit to test the code.

To run the program from the shell, excute these commands:
  1. javac TriangleClassifier.java
  2. java TriangleClassifier


To run the tests from the shell, excute these commands:
  1. javac -cp .:junit-4.12.jar:hamcrest-core-1.3.jar:. Tests.java
  2. java -cp .:junit-4.12.jar:hamcrest-core-1.3.jar org.junit.runner.JUnitCore Tests
